# Project Roadmap - k7-rust-build (Docker Environment)

This roadmap outlines the goals for the next 12 months for the build environment.

## Goals (What we intend to do)
- **Q2 2026:** Automation of weekly base image updates and vulnerability scanning.
- **Q3 2026:** Integration of the latest Rust-for-Linux toolchain stable releases.
- **Q4 2026:** Achievement of OpenSSF Silver Badge certification for the repository.
- **Q1 2027:** Implementation of multi-stage builds to reduce image size and attack surface.

## Non-Goals (What we do NOT intend to do)
- **Legacy Support:** No support for EOL (End of Life) base distributions.
- **Kernel Source Hosting:** This project will not host kernel source code, only the build environment.

# Projekt-Roadmap - k7-rust-build (Docker-Umgebung)

Diese Roadmap skizziert die Ziele für die nächsten 12 Monate für die Build-Umgebung.

## Ziele (Was geplant ist)
- **Q2 2026:** Automatisierung wöchentlicher Base-Image-Updates und Schwachstellen-Scans.
- **Q3 2026:** Integration der neuesten stabilen Rust-for-Linux Toolchain-Releases.
- **Q4 2026:** Erreichung der OpenSSF Silver Badge Zertifizierung für das Repository.
- **Q1 2027:** Implementierung von Multi-Stage-Builds zur Reduzierung der Image-Größe und Angriffsfläche.

## Nicht-Ziele (Was NICHT geplant ist)
- **Legacy-Support:** Keine Unterstützung für EOL (End of Life) Basis-Distributionen.
- **Kernel-Hosting:** Dieses Projekt hostet keinen Kernel-Quellcode, sondern rein die Build-Umgebung.
